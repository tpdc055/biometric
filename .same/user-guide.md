# Citizen Registry - User Guide

## Getting Started

### Installation on Android Phone
1. Open the app URL in Chrome browser
2. You'll see "Add to Home Screen" prompt - tap "Install"
3. The app will be installed like a native app
4. Open from your home screen

### First Time Setup
1. Create a 4-6 digit PIN to protect your data
2. Set up your Recorder Name
3. Add at least one Ward (e.g., "W01 - Mwanakwerekwe")
4. Add Villages under each Ward

## Using the App

### Registering a Household
1. Go to Dashboard > Register Household
2. Select Ward and Village
3. Enter Household Code (e.g., "H001")
4. Enter Head of Household name
5. Optional: Add location description

### Registering a Citizen
1. Go to Dashboard > Register Citizen
2. Select Ward, Village, and Household
3. Capture Photo (tap the camera box)
4. Capture Fingerprint (tap the fingerprint box)
5. Fill in personal details:
   - First Name, Last Name
   - Sex (Male/Female)
   - Date of Birth or Estimated Age
   - Phone Number (optional)
6. Select Occupation and Disability Status
7. Add any notes
8. Confirm consent checkbox
9. Tap "Register"

### Searching Records
1. Go to Dashboard > Search Records
2. Search by name, ID, or phone number
3. Filter by Ward or Village
4. Tap a citizen to view full details
5. Edit or delete from the detail view

### Exporting Data
1. Go to Dashboard > Export Data
2. Select export type (Citizens, Households, or Both)
3. Filter by Ward if needed
4. Choose whether to include photos
5. Tap "Export to CSV"
6. Share the file via WhatsApp, email, etc.

## Offline Mode
- All data is stored locally on your device
- The app works completely offline after first load
- No internet required for registration
- Export files can be shared when online

## Security
- PIN required to access app
- Optional biometric authentication
- All data stored securely in local database
- Never transmitted without your action (export)

## Tips
- Keep your PIN safe - you'll need it every time
- Take clear photos with good lighting
- For fingerprints, use the rear camera and hold finger close
- Export data regularly as backup
